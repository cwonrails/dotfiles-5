# dotfiles

This is where my dotfiles live.

## Installation

```sh
git clone https://github.com/johnotander/dotfiles && cd $_
source bootstrap.sh
```

### Homebrew

```sh
./brew.sh
```

### Npm global packages

```sh
./npm.sh
```

### OS X Defaults

```sh
./.osx
```
